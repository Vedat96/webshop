Hey, dit is mijn website. 
Het is een game webshop.

Gemaakt door Vedat Aydin.

Om de webshop te kunnen openen moet je:
- Het bestand webshop in je htdocs zetten
 (zodat het werkt met XAMP want het is een PHP bestand gelinkt met
database). Daarvoor moet je dus ook XAMP downloaden en in XAMP 
Apache en MySQL aanzetten.
- Het bestand webshop.sql moet je importeren in je MySQL dat doe je 
in http://localhost/phpmyadmin in je webpagina. 
- maak een nieuw database aan in de naam webshop en importeer webshop.sql
- Vervolgens typ je in je browser: localhost/bestandsmap/webshop/public
